What is Bootstrap for Musicians?

Bootstrap for musicians is a toolbox of a few other open source projects, including FitVid.js and AudioPlayer.js by Osvaldas Valutis in order create a simple and effect single page album page for an artist.

Goals

  • Create a single page responsive theme based off of Bootstrap
  • Create custom media queries to adjust text for large displays
  • Address basic UI needs that Bootstrap does not (music player, videos, newsletters etc)
  • Address common meta tags - opengraph (Facebook), schema.org (Google + Bing).
  • Use CDNs when possible (bootstrap, jquery)
  • Create a boilerplate HTML file and a few theme variants
  • Make code as clean and human readable as possible, appropriate for novice web developers and ready for advanced web designers to be able to hack
  • Create a wiki of various services that users may want to add (like addthis, pinterest, vimeo, youtube, gumroad, bandcamp, google calendar, how to create favicons, how to create favicons for mobile devices)

Philosophy

Musician pages often against their better efforts created a bit of a barrier between musicians and their respective fans. While social networking allowed musicians without much web savvy skills interact with fans, they've never really put the music first.

Bootstrap for musicians aims to provide a template that focuses in one specific goal: getting the music to fans. Initially the project will focus on a single album/page design but may expand to, artist profiles.
